# Baseline Reproduction Evidence

This folder contains raw logs and metric JSON files for all reproduced baselines on the Multimodal Mind2Web `test_website` split.

## Files

- `showui_repro.log`: terminal output from ShowUI evaluation.
- `showui_test_website_metrics.json`: parsed metrics from the same run.
- `mindact_repro.log`: terminal output from MindAct evaluation.
- `mindact_test_website_metrics.json`: parsed metrics from the same run.
- `qwen3_*_metrics.json`: metrics for our Qwen3-VL zero-shot modality baselines.

All runs use 1,019 steps and 142 tasks from `test_website`.
